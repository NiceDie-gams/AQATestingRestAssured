Данный репозиторий создан в рамках студенческой практики.
В данном репозитории реализованны API тесты с использование JUnit и RestAssured. 
В качестве API использовался сервис dadata.ru, для работы с ним необходимо зарегистрироваться и получить API-ключ, который после необходимо ввести в .env 
.env.example приложен в репозитории.
